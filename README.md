
# Retail Sales Performance & KPI Analysis using SQL

## Project Overview
This project analyzes retail transaction data using advanced SQL queries.
It evaluates overall sales performance and calculates key business KPIs
to support data-driven decision making.

## Key Objectives
- Analyze total revenue and monthly sales trends
- Calculate Average Order Value (AOV)
- Measure customer purchase frequency
- Identify top-selling products
- Evaluate regional sales performance
- Perform customer segmentation

## SQL Techniques Used
- JOINs
- Subqueries
- GROUP BY
- Window Functions
- Aggregate Functions

## KPIs Calculated
- Total Revenue
- Monthly Sales Trend
- Average Order Value
- Customer Purchase Frequency
- Top Products by Revenue
- Regional Sales Performance
